Mapfeel は、Cosense / Scrapbox のページに書かれた位置情報や写真を地図上で閲覧するためのツールです。

ページに書かれた位置情報タグ（例: [N35,E139]）を読み取り、メモ・写真・記録を地図上で探索できます。 個人の知識や活動記録を「場所」という視点から閲覧することができます。

主な機能

  • Cosense / Scrapbox ページから位置情報タグを取得
  • ページの位置を地図上に表示
  • Gyazo 画像を地図と一緒に閲覧
  • 地理的な記録をまとめて探索
  • 個人の記録や知識を地図として可視化

使い方

  1. Cosense / Scrapbox のページを開きます
  2. 位置情報タグ（例: [N35,E139]）を含むページを表示します
  3. 拡張機能を実行すると、ページの位置情報が地図上に表示されます

こんな用途に

  • 旅行記録・登山記録
  • 写真の位置整理
  • フィールドノート
  • 個人知識管理（PKM）
  • 場所に基づくメモ整理

対応サービス

  • Cosense
  • Scrapbox
  • Gyazo
